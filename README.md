## Summary
This is the backend for my crypto trading practice web API, Cryptiva.
Wtihin this API you can buy/sell crypto, add  your favorite cryptocurrencies to your watchlist, real-time crypto and portfolio updates using SignalR.
My API also handles login and registration, deposit virtual funds and transaction history.

## Stack:
For this API I used: 
-ASP.NET Core Web API
-Angular
-SQL Server & Azure SQL
-CoinGecko API
-Azure App Service

## Improvments:
I am happy with the state of my API but there are things I can improve and add to my API such as:
-Intricate Portfolio Managment
-Admin Dashboard
-Price alerts
-Faster and more intricate updates
